# ColourPalette

**ColourPalette** is a web app that generates customizable color palettes for designers, artists, and developers. You can select the number of colors, enter keywords to inspire palettes, and instantly generate unique combinations. Built with React, TypeScript, Node.js, and MongoDB and integrated OpenAI’s API.

## Features

- **Color Palette Generator:**
  - Choose the number of colors you want in your palette (1-5).
  - Enter keywords (e.g., "vintage", "nature", "bright") to influence the palette generation.
  - Click "GENERATE" to create a new color palette.
  - Each generated palette is previewed instantly onscreen.
- **User History:**
  - Save palettes you generate for reference and download.
  - Easily clear your history with the CLEAR HISTORY button.
- **Randomization:**
  - Create completely random color palettes with the RANDOM button for instant inspiration.
